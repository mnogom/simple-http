# Simple socket server

Use only sockets to create a simple HTTP server.

## Usage
```bash
# run server
./manage.sh run  # or 'r'
# run server as daemon
./manage.sh run_daemon  # or 'rd'
# kill daemon process
./manage.sh kill_daemon  # or 'k'
# ping server (require 'curl')
./manage.sh ping  # or 'p'
```
